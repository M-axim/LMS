# Что это за макет
Автоматизированная система управления школой "LMS" была спроектирована с нуля для курсовой работы

# Особенности макета
* Кроссбраузерность
* Использование API для получения/редактирования данных БД
* Возможность экспорта определенных данных в excel таблицы

# Используемые технологии
* HTML 5 (PUG),
* SCSS,
* Vue.js
* Webpack
* php (laravel)
* Node.js
